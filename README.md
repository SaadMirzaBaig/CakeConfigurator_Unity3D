# CakeConfigurator

A demo web configurator for a 3D model of a cake.


Tasks: 
- Change the measurement of the cake.
- Ability to change the texture. 
- Change the nuts with the raspberry fruits.

Click the link below to try it out!
https://play.unity.com/mg/other/cake-configurator
